**token** *n*
**1** a [[symbol]], [[sign]], or [[indication]] of something: as a token of respect
◇ <small>SYNONYMS</small>: [[symbol]], [[mark]], [[sign]], [[note]], [[expression]], [[indication]], [[representation]], [[badge]]
**2** a gift voucher that can be used as payment for goods of a specified value
**3** a metal or plastic disc, such as a substitute for currency for use in a slot machine
**4** **by the same token** in the same way as something mentioned previously
 *adj*
**5** intended to create an impression but having no real importance: as a token gesture of goodwill
◇ <small>SYNONYMS</small>: [[nominal]], [[symbolic]], [[minimal]], [[hollow]], [[superficial]], [[perfunctory]]
