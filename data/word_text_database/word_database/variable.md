**variable** *adj*
**1** likely to change at any time: variable weather
◇ <small>SYNONYMS</small>: [[changeable]], [[unstable]], [[fluctuating]], [[shifting]], [[flexible]], [[uneven]], [[temperamental]], [[unsteady]]
**2** (*maths*) having a range of possible values
 *n*
**3** something that is subject to variation
**4** (*maths*) an expression that can be assigned any of a set of values
 **[[variability]]** *n* 
 **[[variably]]** *adv*
