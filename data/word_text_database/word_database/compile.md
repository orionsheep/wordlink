**compile** *vb* **-piling**, **-piled**
**1** to [[collect]] and arrange (information) from various sources
◇ <small>SYNONYMS</small>: [[put together]], [[collect]], [[gather]], [[organize]], [[accumulate]], [[marshal]], [[garner]], [[amass]]
**2** (*computers*) to convert (commands for a computer) from the language used by the person using it into machine code suitable for the computer, using a compiler
 **[[compilation]]** *n*
