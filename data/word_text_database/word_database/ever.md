**ever** *adv*
**1** [[at any time]]: it was the fourth fastest time ever
◇ <small>SYNONYMS</small>: [[at any time]], [[at all]], [[in any case]], [[at any point]], [[by any chance]], [[on any occasion]], [[at any period]]
**2** [[always]]: ever present
◇ <small>SYNONYMS</small>: [[always]], [[for ever]], [[at all times]], [[evermore]]
**3** used to give emphasis: tell him to put to sea as soon as ever he can
**4** **ever so** *or ***ever such** (*informal*, *chiefly* *Brit*) used to give emphasis: I'm ever so sorry
