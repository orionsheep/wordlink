**set up** *vb*
**1** to build or construct: the soldiers had actually set up a munitions factory
◇ <small>SYNONYMS</small> **[[set something up]]**: build, raise, construct, put up, assemble, put together, erect
**2** to put into a position of power or wealth
**3** to begin or enable (someone) to begin (a new venture): he set up a small shop
◇ <small>SYNONYMS</small> **[[set something up]]**: establish, begin, found, institute, initiate
**4** to begin or produce: to set up a nuclear chain reaction
**5** to establish: Broad set up a world record
**6** (*informal*) to cause (a person) to be blamed or accused
**7** to restore the health of: a pub lunch set me up nicely
 *n* **setup**
**8** (*informal*) the way in which anything is organized or arranged
◇ <small>SYNONYMS</small>: [[arrangement]], [[system]], [[structure]], [[organization]], [[conditions]], [[regime]]
**9** (*slang*) an event the result of which is prearranged
